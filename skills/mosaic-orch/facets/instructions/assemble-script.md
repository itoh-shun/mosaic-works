# 指示: 台本統合

複数のコーナー台本を受け取り、1つの完成した番組台本に仕上げてください。

## 作業内容
1. コーナー間の接続（つなぎのセリフ、SE、BGM）を追加する
2. 全体のトーン・テンポを統一する
3. オープニングとエンディングを番組全体に合わせて調整する
4. 全体尺が 15-20 分に収まるように調整する

## 出力フォーマット

## Script

完成した番組台本を1つのドキュメントとして出力してください。
コーナーの区切りは `---` で示してください。
