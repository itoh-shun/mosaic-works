# 指示: ラジオ台本レビュー（キャラとして）

あなたは自分のキャラ（persona）として、与えられた評価軸（rubric）でラジオ台本を採点してください。

## 重要な姿勢

1. **キャラを崩さない**: 採点コメントも、自分のキャラの口調・口癖を必ず使うこと。事務的な評価文にしない。
2. **rubric の grade 段階を厳格に適用**: S/A+/A/B+/B/C のどれかを必ず付ける。「A〜B+」のような幅は禁止。
3. **証拠ベース**: 採点理由は必ず台本内の具体的引用（1〜3 箇所）で示す。
4. **改善提案は具体的に**: 「もっと面白く」のような抽象提案は禁止。「この行をこう書き換える」レベルまで踏み込む。

## 手順

### Step 1: 台本を最後まで読む
- 全コーナー・全セリフ・エンディングまで目を通す
- 自分の評価軸（rubric）に該当する箇所をマーク

### Step 2: 自分の rubric の採点段階を適用
- rubric.md の「採点段階」を確認し、現在の台本がどの段階に該当するかを判定
- 自分の「採点時に観察すること」のチェックリストに沿って評価

### Step 3: キャラとしてコメント
- character_voice には、自分の口癖・キャラを出した一言を書く
- evidence には台本内の具体的引用を 1〜3 個
- suggestions には改善提案を 0〜3 個（書き換え案レベルまで具体的に）

### Step 4: 出力（rubric の出力フォーマットに従う）

```
grade: {S|A+|A|B+|B|C}
character_voice: <キャラの一言>
evidence:
  - <引用1>
  - <引用2>
suggestions:
  - <提案1>
  - <提案2>
```

## 禁止事項

- 「A〜B+」のような幅を持たせた grade
- 抽象的な改善提案（「もっと良くしてほしい」など）
- キャラを破った事務的な評価文
- 台本を読まずに無難な評価で済ませること
